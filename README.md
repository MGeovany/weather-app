## 🚀 Get Started
  
You need to be using:

- [Git](https://git-scm.com/downloads)

- [Nodejs v16 or higher](https://nodejs.org/es/download/)

  

Clone the repository & install dependencies:

  

```bash

git clone https://github.com/eriandev/weather-app.git

cd weather-app

npm install

```
